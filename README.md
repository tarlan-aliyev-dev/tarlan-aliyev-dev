### About

I'm a software engineer with a QA foundation, now building deep in AI. I design **agentic systems and developer tools on top of LLMs** — from a personal JARVIS-style command center to LLM-driven QA copilots. I care about dependable automation, interfaces that feel effortless, and tools people actually reach for.

- 🌱 Exploring how agentic workflows reshape testing and everyday knowledge work
